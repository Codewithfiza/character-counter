# ✍️ TypeCount — Character Counter

A warm, editorial-styled character counter built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- Real-time character counting as you type
- Line counter that tracks each new line
- Clear button to reset everything instantly
- Elegant slide-up card animation on load
- Lined paper textarea with corner bracket accents
- Newspaper / typewriter aesthetic

## 🛠️ Built With

- HTML5
- CSS3 (animations, CSS variables, paper texture)
- Vanilla JavaScript (no libraries)

## 📂 Project Structure

```
character-counter/
├── index.html
├── style.css
└── script.js
```

## 🚀 How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Start typing in the textarea
4. Watch the character and line count update live
5. Hit **Clear Text** to reset

## 💡 Concepts Applied

- DOM manipulation with `getElementById`
- Event listeners (`input`, `click`)
- String `.length` property for character counting
- String `.split("\n")` for line counting
- Conditional logic for edge cases
- Dynamic `textContent` updates
